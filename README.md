# solar_engine

## Introduction :

A galgame engine made by cmach_socket.

Aiming be a cross-platform engine , including desktop and mobile platforms ( Yes you can lie down and play it just on your phone ), it also consider of the performance usage.

Also, It target to creat a simple sceario file standard , owning completely compileless explain ablibity,through enormous custom settings  to make up cross-platform diffrence, to give better stage performance.

Powered by flutter/dart.
